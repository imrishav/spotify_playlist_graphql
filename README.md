# For Running The Project

In Server Directory

1.  Add client id and secret from spotify,
2.  Then run npm install
3.  and npm start
4.  Then you can navigate to localhost:4000/graphql

In Frontend

1.npm install and then npm start

### Technologies used

1.React
2.React Context API
3.Styled Components
4.React Router-dom
5.Graphql,Apollo
6.React Slick
7.Express(apollo express server)
